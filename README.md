# Portfolio Project

A personal portfolio static website showcasing projects, skills and courses. Designed with a modern, responsive layout and subtle animated effects.

## Features
- Responsive layout for desktop, tablet and mobile
- Project cards with popup details
- Tech stack section with tooltips
- Animated gradient background and subtle UI effects
- Footer with social links

## Tech Stack
- HTML5
- CSS3 (custom styles, animations)
- Optional: JavaScript for interactive popups (js/main.js)

## Local development
1. Open index.html in a browser, or run a simple local server:
   - Using Python (Windows):
     ```
     python -m http.server 8000
     ```
     Then open http://localhost:8000
   - Or use VS Code Live Server extension.

2. Edit files in:
   - c:\Users\ASUS\Downloads\My-portfolio-website\index.html
   - c:\Users\ASUS\Downloads\My-portfolio-website\css\master.css
   - c:\Users\ASUS\Downloads\My-portfolio-website\js\main.js

## Deployment
- Recommended: GitHub Pages.
  - If repository is named `<username>.github.io`, push to `main` (or `master`) and enable Pages in repo settings.
  - Or deploy with a `gh-pages` branch / GitHub Actions.
